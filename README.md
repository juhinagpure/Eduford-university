## Installation
1. Clone the repository:
   ```
   git clone https://github.com/juhinagpure/Eduford-university.git
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open the application in your browser:
   ```
   http://localhost:3000
   ```
